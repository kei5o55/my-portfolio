### ポートフォリオサイト
個人制作のポートフォリオサイトです。  
制作アプリ・イラスト作品・世界観プロジェクトをまとめて展示するために制作しました。

Next.js（App Router）を使ってサイト構造を作

## Demo

https://kei5ot.com/

---

## 内容

- トップページ / 作品一覧 / 作品詳細ページ
- イラスト作品のタグ付き一覧
- クリックで拡大表示できるギャラリー
- TypeScript + React
- Next.js App Router

---
## 今後実装したい機能
- ブログ機能
- 問い合わせフォーム

## 仕様技術

- Next.js
- React
- TypeScript
- CSS (inline + custom styles)
